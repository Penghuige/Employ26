## 📦 模块说明

### 1. preprocessing/ - 数据预处理
**功能**: 原始数据的清洗、采样、分割

**文件**:
- `sample_data.py` - 从原始数据中均匀采样（10%）
- `split.py` - 将大文件分割成小文件

**使用**:
```bash
# 提取样本
python -m src.preprocessing.sample_data

# 分割数据
python -m src.preprocessing.split
```

---

### 2. nlp_analysis/ - NLP分析
**功能**: 文本清洗、分词、关键词提取

**文件**:
- `text_preprocessing.py` - 完整的NLP预处理流程
  - 文本清洗（去HTML、URL、特殊字符）
  - 中文分词（jieba + 自定义词典）
  - 停用词过滤
  - 关键词提取（TF-IDF）
  
- `quick_nlp.py` - 快速NLP处理（小样本）

**使用**:
```bash
# 完整NLP处理
python -m src.nlp_analysis.text_preprocessing

# 快速处理
python -m src.nlp_analysis.quick_nlp
```

---

### 3. analysis/ - 数据分析
**功能**: 薪资、技能、趋势等多维度分析

**文件**:
- `salary_analysis.py` - 薪资分析
  - 薪资统计（平均、中位数、分位数）
  - 技能薪资排行
  - 学历薪资差异
  - 经验薪资差异
  - 城市薪资排行
  
- `skill_combination.py` - 技能组合分析
  - 技能对共现统计
  - 技能三元组分析
  - 技能搭配推荐
  - 技能关系网络图
  
- `time_trend_analysis.py` - 时间趋势分析
  - 招聘量趋势（年度、季度、月度）
  - 技能需求趋势
  - 热门岗位趋势

**使用**:
```bash
# 薪资分析
python -m src.analysis.salary_analysis

# 技能组合分析
python -m src.analysis.skill_combination

# 时间趋势分析
python -m src.analysis.time_trend_analysis
```

---

### 4. visualization/ - 可视化
**功能**: 生成词云、图表等可视化内容

**文件**:
- `wordcloud_generator.py` - 词云生成
  - 技能关键词词云
  - 热门岗位词云
  - 行业分布词云
  - 生成HTML交互式词云

**使用**:
```bash
# 生成词云
python -m src.visualization.wordcloud_generator
```

---

### 5. utils/ - 工具函数
**功能**: 辅助工具和检查脚本

**文件**:
- `analyze_results.py` - 分析结果统计
  - 关键词统计
  - 技能需求统计
  - 城市分布统计
  
- `check_samples.py` - 样本数据检查
  - 查看样本文件信息
  - 数据质量检查

**使用**:
```bash
# 分析结果
python -m src.utils.analyze_results

# 检查样本
python -m src.utils.check_samples
```

## 📝 注意事项

1. **运行脚本时的路径**
   - 主运行脚本在根目录，直接运行即可
   - 模块内的脚本使用 `python -m` 方式运行

2. **导入路径**
   - 所有导入使用 `from src.xxx import yyy` 格式
   - 确保在项目根目录运行脚本

3. **数据文件**
   - 原始数据在 `data/` 目录
   - 处理结果在 `output/` 目录
   - 不要将大文件提交到git

4. **词典文件**
   - 自定义词典在 `dicts/` 目录
   - NLP模块会自动加载
