# 技能识别方法说明与改进方案

## 📋 当前技能识别方法

### 1. 识别流程

当前的技能识别采用**基于词典匹配 + TF-IDF关键词提取**的混合方法：

#### 步骤1：自定义词典加载
- 位置：`dicts/userdict_zh_recruitment.txt`
- 包含约**180+个**技能词汇
- 使用jieba分词的`add_word()`方法，设置高频率权重（10000-20000）
- 确保这些技能词不会被错误分词

#### 步骤2：文本预处理
```python
# 清洗文本
- 去除HTML标签
- 去除URL链接
- 去除邮箱
- 去除特殊字符
- 统一空白字符

# 中文分词
- 使用jieba分词
- 过滤停用词（通用词 + 招聘领域词）
- 过滤长度<2的短词
```

#### 步骤3：关键词提取
```python
# 使用jieba.analyse.extract_tags()
- 算法：TF-IDF
- 每个岗位提取Top 10关键词
- 自动识别重要的技能词汇
```

#### 步骤4：技能匹配
```python
# 在分析阶段
- 定义目标技能列表（约50个核心技能）
- 在关键词中搜索是否包含目标技能
- 统计技能出现频率和薪资关系
```

## ⚠️ 当前方法的局限性

### 1. **覆盖不全面**
**问题：**
- 自定义词典只有180+词汇，实际技能可能有500+
- 目标技能列表只有50个，遗漏了很多细分技能
- 新兴技能（如RAG、LoRA、Prompt工程）覆盖不足

**影响：**
- 无法识别长尾技能
- 新技术趋势可能被忽略
- 细分领域分析不够深入

### 2. **识别不够精准**
**问题：**
- TF-IDF可能提取到非技能词汇（如"负责"、"完成"）
- 没有区分技能类型（编程语言 vs 框架 vs 工具 vs 软技能）
- 同义词未合并（如"机器学习"和"ML"、"人工智能"和"AI"）

**影响：**
- 统计结果有噪音
- 技能分类不清晰
- 重复统计同一技能

### 3. **缺少语义理解**
**问题：**
- 纯粹基于字符串匹配
- 无法理解上下文（"熟悉Python" vs "不需要Python"）
- 无法识别技能的重要程度（"精通" vs "了解"）

**影响：**
- 可能误判技能要求
- 无法区分核心技能和加分项
- 技能熟练度要求无法量化

### 4. **技能层级不清晰**
**问题：**
- 所有技能平等对待
- 没有技能分类体系（前端/后端/数据/运维等）
- 没有技能依赖关系（学Python前需要先学编程基础）

**影响：**
- 难以构建技能树
- 职业路径规划困难
- 技能推荐不够智能

#### 2.1 多算法融合
```python
def extract_skills_advanced(text):
    """改进的技能提取"""
    skills = set()
    
    # 方法1：词典匹配（精准）
    for skill in skill_dictionary:
        if skill in text:
            skills.add(normalize_skill(skill))
    
    # 方法2：TF-IDF（发现新技能）
    tfidf_keywords = extract_tfidf_keywords(text, topK=20)
    for keyword in tfidf_keywords:
        if is_skill_like(keyword):  # 判断是否像技能词
            skills.add(keyword)
    
    # 方法3：正则表达式（特定模式）
    # 匹配"熟悉XX"、"精通XX"、"掌握XX"
    patterns = [
        r'熟悉(\w+)',
        r'精通(\w+)',
        r'掌握(\w+)',
        r'了解(\w+)',
        r'使用(\w+)',
    ]
    for pattern in patterns:
        matches = re.findall(pattern, text)
        skills.update(matches)
    
    return list(skills)
```

```python
def extract_skill_with_level(text):
    """提取技能及熟练度"""
    skill_levels = []
    
    level_patterns = {
        '精通': 5,
        '熟练': 4,
        '熟悉': 3,
        '了解': 2,
        '接触': 1
    }
    
    for level_word, level_score in level_patterns.items():
        pattern = f'{level_word}(\\w+)'
        matches = re.findall(pattern, text)
        for skill in matches:
            skill_levels.append({

                'level': level_score,
                'level_text': level_word
            })
    
    return skill_levels
```

#### 3.2 基于BERT的NER（高级）
```python
from transformers import AutoTokenizer, AutoModelForTokenClassification
import torch

def extract_skills_bert(text):
    """使用BERT模型提取技能"""
    # 加载预训练的中文NER模型
    model_name = "ckiplab/bert-base-chinese-ner"
    tokenizer = AutoTokenizer.from_pretrained(model_name)
    model = AutoModelForTokenClassification.from_pretrained(model_name)
    
    # 分词和预测
    inputs = tokenizer(text, return_tensors="pt")
    outputs = model(**inputs)
    predictions = torch.argmax(outputs.logits, dim=2)
    
    # 解析实体
    tokens = tokenizer.convert_ids_to_tokens(inputs["input_ids"][0])
    labels = predictions[0].tolist()
    
    skills = []
    current_skill = []
    
    for token, label in zip(tokens, labels):
        if label == 1:  # B-SKILL
            if current_skill:
                skills.append(''.join(current_skill))
            current_skill = [token]
        elif label == 2:  # I-SKILL
            current_skill.append(token)
        else:
            if current_skill:
                skills.append(''.join(current_skill))
                current_skill = []
    
    return skills
```

```python
def calculate_skill_importance(skill, job_description):
    """计算技能重要性"""
    score = 0
    
    # 1. 出现位置（越靠前越重要）
    position = job_description.find(skill)
    if position != -1:
        position_score = 1 - (position / len(job_description))
        score += position_score * 20
    
    # 2. 出现次数
    frequency = job_description.count(skill)
    score += min(frequency * 10, 30)
    
    # 3. 修饰词（精通>熟练>熟悉>了解）
    modifiers = {
        '精通': 30,
        '熟练': 25,
        '熟悉': 20,
        '了解': 10,
        '优先': 15
    }
    for modifier, modifier_score in modifiers.items():
        if f'{modifier}{skill}' in job_description:
            score += modifier_score
    
    # 4. 是否在标题中
    if skill in job_title:
        score += 40
    
    # 5. 市场薪资权重
    skill_salary = get_skill_avg_salary(skill)
    overall_salary = get_overall_avg_salary()
    salary_ratio = skill_salary / overall_salary
    score += (salary_ratio - 1) * 20
    
    return min(score, 100)
```

## 💻 代码示例：改进的技能提取器

```python
class AdvancedSkillExtractor:
    """改进的技能提取器"""
    
    def __init__(self):
        self.skill_dict = self.load_skill_dictionary()
        self.skill_taxonomy = self.load_taxonomy()
        self.synonyms = self.load_synonyms()
    
    def extract(self, text):
        """提取技能"""
        skills = []
        
        # 1. 词典匹配
        dict_skills = self.extract_by_dictionary(text)
        skills.extend(dict_skills)
        
        # 2. 模式匹配
        pattern_skills = self.extract_by_pattern(text)
        skills.extend(pattern_skills)
        
        # 3. TF-IDF
        tfidf_skills = self.extract_by_tfidf(text)
        skills.extend(tfidf_skills)
        
        # 4. 去重和标准化
        skills = self.normalize_skills(skills)
        
        # 5. 分类
        categorized_skills = self.categorize_skills(skills)
        
        return categorized_skills
    
    def extract_with_importance(self, text, title):
        """提取技能及重要性"""
        skills = self.extract(text)
        
        skill_scores = []
        for skill in skills:
            score = self.calculate_importance(skill, text, title)
            skill_scores.append({
                'skill': skill,
                'importance': score,
                'category': self.get_category(skill)
            })
        
        return sorted(skill_scores, key=lambda x: x['importance'], reverse=True)
```
