# Cursor配置文件说明

## 为什么创建这些配置文件

### 1. `.cursorrules` - Cursor规则文件

**作用**：
- 定义项目的编码规范和最佳实践
- 指导AI助手生成符合项目要求的代码
- 统一团队开发风格

**为什么需要**：
- ✅ **大数据处理规范**：5GB数据必须使用分块处理，避免内存溢出
- ✅ **NLP处理规范**：统一中文分词、停用词过滤等流程
- ✅ **性能优化指南**：并行计算、缓存机制等
- ✅ **代码质量保证**：异常处理、日志记录、注释规范
- ✅ **项目结构约定**：统一目录结构，便于协作

**核心内容**：
1. 项目概述和目标
2. 数据字段说明
3. Python编码规范
4. 数据处理原则（分块、增量、并行、缓存）
5. NLP处理规范
6. 性能优化策略
7. 项目结构约定
8. 技术栈说明
9. AI辅助开发指南

**使用场景**：
- 当你让AI生成代码时，它会自动遵循这些规则
- 例如：生成数据处理代码时，会自动使用chunksize分块读取
- 例如：生成NLP代码时，会自动使用jieba分词

---

### 2. `.cursor/skills.md` - 技能库

**作用**：
- 提供常用代码片段和最佳实践
- 作为代码模板库，快速复用
- 记录项目特定的技术方案

**为什么需要**：
- ✅ **提高开发效率**：不用每次都从零写代码
- ✅ **保证代码质量**：使用经过验证的代码模板
- ✅ **知识沉淀**：记录项目中的技术方案
- ✅ **团队协作**：统一技术实现方式

**包含的技能**：
1. **数据处理技能**
   - 大文件分块读取
   - 并行处理
   - 内存优化

2. **NLP技能**
   - 中文文本预处理
   - 技能关键词提取
   - 主题建模
   - 词向量训练

3. **薪资分析技能**
   - 薪资范围解析
   - 薪资预测模型

4. **可视化技能**
   - 词云生成
   - 交互式图表

5. **性能监控技能**
   - 内存监控
   - 进度条

**使用场景**：
- 需要实现某个功能时，先查看skills.md是否有现成模板
- AI可以参考这些模板生成更符合项目需求的代码
- 新成员加入时，快速了解项目的技术实现

---

### 3. `.cursor/commands.md` - 常用命令

**作用**：

- ✅ **流程文档化**：记录完整的分析流程

**包含的命令**：

   - 技能图谱构建

3. **数据分析命令**
   - 薪资分析
   - 趋势分析
   - 地域分析

4. **可视化命令**
   - 生成词云
   - 生成报告

5. **模型评估命令**

## 三个文件的关系

```
.cursorrules (规则)
    ↓ 指导
.cursor/skills.md (实现)
    ↓ 封装
.cursor/commands.md (使用)
```

- **`.cursorrules`**：定义"应该怎么做"（规范）
- **`skills.md`**：提供"怎么实现"（代码模板）
- **`commands.md`**：说明"怎么使用"（操作手册）

## 实际使用示例

### 场景1：处理大文件

**1. 查看规则（.cursorrules）**
```
数据处理原则：
- 分块处理：5GB数据必须使用chunk方式读取
```

**2. 查看技能（skills.md）**
```python
def read_large_csv_chunks(filepath, chunksize=10000):
    for chunk in pd.read_csv(filepath, chunksize=chunksize):
        yield chunk
```

**3. 查看命令（commands.md）**
```bash
python src/preprocessing/clean_data.py --input data/ --output output/processed/
```

**4. AI生成代码**
当你说"帮我处理这个5GB的CSV文件"，AI会：
- 根据`.cursorrules`知道要用分块处理
- 参考`skills.md`的代码模板
- 生成符合规范的代码

---

### 场景2：提取技能关键词

**1. 查看规则（.cursorrules）**

```
NLP处理规范：

- 停用词过滤必须包含招聘领域特定词汇
```

**2. 查看技能（skills.md）**

```python
    vectorizer = TfidfVectorizer(max_features=1000)
    tfidf_matrix = vectorizer.fit_transform(job_descriptions)
```

## 如何使用这些配置

### 在Cursor中
1. **自动生效**：`.cursorrules`会自动被Cursor读取
2. **引用技能**：可以在对话中说"参考skills.md中的分块读取方法"
3. **执行命令**：可以直接从commands.md复制命令执行

### 在开发中
1. **开始新功能前**：先查看是否有相关规范和模板
2. **遇到问题时**：查看是否有类似的解决方案
3. **代码审查时**：对照规范检查代码质量
4. **文档更新**：发现新的最佳实践及时补充
