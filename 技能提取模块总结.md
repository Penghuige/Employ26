### 2. Word2Vec技能扩展器 ✅

**文件：** `word2vec_extractor.py`

**功能：**
- ✅ 从NLP处理后的数据训练Word2Vec模型
- ✅ 基于50个种子技能查找相似技能
- ✅ 支持样本数据验证（use_sample=True）
- ✅ 自动保存模型到 `models/` 目录
- ✅ 生成jieba词典格式
- ✅ 生成详细分析报告

**核心参数：**
```python
vector_size = 300      # 词向量维度
window = 5             # 上下文窗口
min_count = 5          # 最小词频
epochs = 10            # 训练轮数
topn = 20              # 每个技能查找top-n相似词
threshold = 0.5        # 相似度阈值
```

**输出文件：**
- `word2vec_expanded_skills.txt` - jieba词典格式
- `word2vec_expanded_skills.json` - 详细结果
- `word2vec_expansion_report.txt` - 分析报告
- `models/word2vec_skills.model` - 训练的模型

### 3. BERT命名实体识别器 ✅

**文件：** `bert_extractor.py`

**功能：**
- ✅ 使用BERT-NER模型提取技能实体
- ✅ 自动检测GPU并使用（如果可用）
- ✅ 批量处理，支持进度显示
- ✅ 支持样本数据和行数限制（快速测试）
- ✅ 统计技能出现频率
- ✅ 生成多种格式的输出

**核心参数：**
```python
model_name = "ckiplab/bert-base-chinese-ner"
max_length = 512       # 最大文本长度
batch_size = 100       # 批处理大小
score_threshold = 0.5  # 置信度阈值
max_rows = 1000        # 快速测试时的最大行数
```

**输出文件：**
- `bert_extracted_skills.txt` - jieba词典格式
- `bert_extracted_entities.json` - 详细提取结果
- `bert_skill_statistics.csv` - 技能统计表
- `bert_extraction_report.txt` - 分析报告

### 4. 整合流水线 ✅

**文件：** `run_extraction_pipeline.py`

**功能：**
- ✅ 整合Word2Vec和BERT两种方法
- ✅ 自动运行完整流程
- ✅ 合并两种方法的结果
- ✅ 记录技能来源（seed/w2v/bert）
- ✅ 生成最终词典和对比报告

**流程：**
```
阶段1: Word2Vec快速扩展
  ↓
阶段2: BERT命名实体识别
  ↓
阶段3: 合并结果
  ↓
阶段4: 保存最终词典
```

**输出文件：**
- `final_expanded_skills.txt` - 最终合并词典
- `final_skills_detail.json` - 详细信息（含来源）
- `final_extraction_report.txt` - 对比报告

## 📋 使用指南

### 快速开始

```bash
# 1. 安装基础依赖（Word2Vec）
pip install pandas numpy gensim tqdm

# 2. 运行Word2Vec（不需要GPU）
python src/skill_extraction/word2vec_extractor.py

# 3. 安装BERT依赖（可选）
pip install transformers torch

# 4. 运行完整流水线
python src/skill_extraction/run_extraction_pipeline.py
```

### 在代码中使用

```python
from src.skill_extraction import SkillExtractionPipeline

# 创建流水线
pipeline = SkillExtractionPipeline()

# 运行完整流程
all_skills = pipeline.run_full_pipeline(
    use_sample=True,   # 使用样本数据
    max_rows=1000      # BERT处理1000行
)
```
