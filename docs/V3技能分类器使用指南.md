# V3技能分类器使用指南

## 🎯 V3方案概述

V3使用**机器学习分类器**自动判断新词是否为技能，无需人工维护完整词典。

### 核心技术

1. **特征工程**（125维特征）
   - 基础特征：长度、大小写、是否含数字等
   - 词性特征：名词、动词、形容词等
   - 词向量特征：Word2Vec 100维向量
   - 相似度特征：与种子词的相似度统计
   - 统计特征：文档频率、TF-IDF等

2. **随机森林分类器**
   - 100棵决策树
   - 自动学习技能vs非技能的模式
   - 输出置信度（0-1）

3. **主动学习**
   - 从种子词典学习正样本
   - 从黑名单学习负样本
   - 可持续更新模型

## 🔧 使用方法

### 方法1：批量发现新技能

```python
from src.skill_extraction.v3_skill_classifier import SkillClassifierV3

# 创建分类器
classifier = SkillClassifierV3()

# 运行流水线（自动训练+发现）

    use_sample=True,
    max_rows=5000
)

# 查看结果

```

### 方法2：预测单个词

```python
# 加载已训练的分类器
classifier = SkillClassifierV3()
classifier.load_word2vec_model()

# 加载分类器模型
import pickle
with open('models/skill_classifier_v3.pkl', 'rb') as f:
    classifier.skill_classifier = pickle.load(f)

# 预测
is_skill, confidence, category = classifier.predict_skill('Rust')

if is_skill:
    print(f"✅ Rust 是技能")
    print(f"   置信度: {confidence:.3f}")
    print(f"   类别: {category}")
else:
    print(f"❌ Rust 不是技能")
    print(f"   原因: {category}")
```

### 方法3：批量预测

```python
test_words = ['Rust', 'Golang', 'TypeScript', '沟通能力', '负责']
```

## 📋 特征说明

### 125维特征向量

```python
特征组成：
├── 基础特征（5维）
│   ├── 长度
│   ├── 是否全大写
│   ├── 是否全小写
│   ├── 是否含数字
│   └── 是否全字母
│
├── 词性特征（10维 one-hot）
│   ├── 名词(n)
│   ├── 动词(v)
│   ├── 形容词(a)
│   ├── 副词(d)
│   ├── 英文(eng)
│   ├── 专有名词(nz)
│   ├── 名动词(vn)
│   ├── 名形词(an)
│   ├── 人名(nr)
│   └── 其他(other)
│
├── 词向量特征（100维）
│   └── Word2Vec向量
│
├── 相似度特征（5维）
│   ├── 与种子词平均相似度
│   ├── 与种子词最大相似度
│   ├── 与种子词最小相似度
│   ├── 相似度标准差
│   └── 高相似度(>0.7)数量
│
└── 统计特征（5维）
    ├── 文档频率
    ├── 文档频率比例
    ├── 总频率
    ├── 平均每文档出现次数
    └── TF-IDF近似值
```

## 🔄 工作流程

### 完整流程

```
1. 加载数据
   ↓
2. 训练Word2Vec模型（如果不存在）
   ↓
3. 收集统计信息（文档频率、总频率）
   ↓
4. 准备训练数据
   - 正样本：种子词典
   - 负样本：黑名单
   ↓
5. 提取特征（125维）
   ↓
6. 训练随机森林分类器
   ↓
7. 评估模型性能
   ↓
8. 发现新技能
   - 收集候选词
   - 过滤低频词
   - 预测每个候选词
   - 过滤低置信度
   ↓
9. 保存结果
```

## 🐛 常见问题

### Q1: 训练时间太长？

**A**: 减少数据量或降低模型复杂度

```python
# 使用更少数据
classifier.run_pipeline(use_sample=True, max_rows=1000)

# 减少树的数量
n_estimators=50  # 默认100
```

### Q2: 发现的新技能质量不高？

**A**: 提高置信度阈值或增加训练样本

```python
# 提高阈值
threshold=0.90  # 默认0.85

# 增加负样本（编辑黑名单）
# 增加正样本（编辑种子词典）
```

### Q3: 某些明显的技能没被识别？

**A**: 检查是否在黑名单中，或者相似度特征不足

```python
# 检查词是否在黑名单
is_black, black_type = classifier.dict_manager.is_blacklisted('XXX')

# 检查词向量
if 'XXX' in classifier.word2vec_model.wv:
    print("词在模型中")
else:
    print("词不在模型中，无法提取词向量特征")
```

### Q4: 如何更新模型？

**A**: 更新种子词典和黑名单后重新训练

```bash
# 1. 更新词典
# 编辑 dicts/skill_seeds.txt
# 编辑 dicts/blacklist_*.txt

# 2. 重新训练
python test_v3_classifier.py
```

## 📚 技术细节

### 为什么选择随机森林？

1. **鲁棒性好**：对噪声和异常值不敏感
2. **特征重要性**：可以分析哪些特征最重要
3. **无需特征缩放**：不需要标准化
4. **并行训练**：速度快
5. **过拟合风险低**：集成学习

### 特征工程的考虑

1. **词向量**：捕捉语义信息
2. **相似度**：利用已知技能的知识
3. **统计特征**：捕捉使用模式
4. **词性特征**：技能通常是名词
5. **基础特征**：简单但有效的规则
