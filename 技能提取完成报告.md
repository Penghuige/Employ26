### 1. 使用Word2Vec发现相似技能 ✅
**文件：** `src/skill_extraction/word2vec_extractor.py`

**功能实现：**
- ✅ 从分词后的岗位描述训练Word2Vec模型
- ✅ 基于50个种子技能查找相似技能
- ✅ 支持相似度阈值过滤（默认0.5）
- ✅ 自动保存模型供后续使用
- ✅ 生成jieba词典格式和详细报告

**核心代码：**
```python
# 训练Word2Vec
model = Word2Vec(
    sentences,
    vector_size=300,
    window=5,
    min_count=5,
    epochs=10,
    sg=1  # Skip-gram
)

# 查找相似技能
similar = model.wv.most_similar('Python', topn=20)
```

**文件：** `src/skill_extraction/bert_extractor.py`

**功能实现：**
- ✅ 使用BERT-NER模型提取技能实体
- ✅ 自动检测GPU并使用（如果可用）
- ✅ 批量处理，支持进度显示

- ✅ 生成多种格式输出

**核心代码：**
```python
# 加载BERT-NER模型
from transformers import pipeline

ner_pipeline = pipeline(
    "ner",
    model="ckiplab/bert-base-chinese-ner",
    device=0  # GPU

entities = ner_pipeline(text)
```
