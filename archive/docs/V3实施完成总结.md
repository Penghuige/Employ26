## 🎯 V3核心技术

### 1. 特征工程（125维）

```python
特征组成：
├── 基础特征（5维）：长度、大小写、是否含数字等
├── 词性特征（10维）：名词、动词、形容词等
├── 词向量特征（100维）：Word2Vec语义向量
├── 相似度特征（5维）：与种子词的相似度统计
└── 统计特征（5维）：文档频率、TF-IDF等
```

### 2. 随机森林分类器

```python
RandomForestClassifier(

    max_depth=20,          # 最大深度20
    min_samples_split=5,   # 分裂最小样本5
    random_state=42
)
```

## 💡 组合使用策略

### 推荐方案：两阶段流水线

```python
# 阶段1：MVP抽取已知技能（高精度90%）
mvp_skills = mvp_extractor.extract_from_dataset(df)

# 阶段2：V3发现新技能（高召回75-80%）
v3_skills = v3_classifier.discover_new_skills(texts)

# 合并结果
all_skills = {**mvp_skills, **v3_skills}
```

**优势**：
- ✅ 已知技能用MVP（90%精度）
- ✅ 新技能用V3发现
- ✅ 综合精度和召回率都高
